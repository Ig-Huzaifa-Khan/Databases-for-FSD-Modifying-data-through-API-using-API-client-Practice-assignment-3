User Registration System

Backend API for secure user registration with validation and password hashing

API Endpoints

POST /api/register

Register a new user

{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "securepassword123"
}

GET /api/users

Get all registered users (for testing only)

GET /health

Check server health status

Features

  • ✅ Secure password hashing with bcrypt
  • ✅ Email format validation
  • ✅ Password minimum length validation (6 characters)
  • ✅ Duplicate email prevention
  • ✅ Comprehensive error handling
  • ✅ RESTful API design

Testing with Bruno/Postman

Base URL: http://localhost:3010

Use Bruno or Postman to test the API endpoints with the JSON payload above.
